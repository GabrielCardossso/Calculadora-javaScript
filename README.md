# ✨ Funcionalidades

✅ Operações matemáticas básicas
✅ Suporte ao teclado
✅ Suporte a números decimais
✅ Backspace funcional
✅ Tratamento de operações inválidas
✅ Interface organizada em layout 4x4
✅ Projeto totalmente sem uso de `eval()`
✅ Manipulação manual das operações matemáticas

---

# 🧠 Como funciona

A calculadora foi desenvolvida utilizando JavaScript puro, sem bibliotecas externas.

O projeto implementa:

* captura de eventos do teclado
* manipulação dinâmica do display
* controle manual das operações
* validações matemáticas
* tratamento de erros de entrada

# 📈 Evolução do projeto

## 🔹 V1.0 – Versão inicial

* Estrutura básica em HTML
* Calculadora funcional
* Operações simples
* Sem preocupação com estética

---

## 🔹 V2.0 – Organização do código

* Separação de responsabilidades
* Melhor estrutura do JavaScript
* Pequenos ajustes de funcionamento

---

## 🔹 V3.0 – Remoção do eval()

* Removido o uso de `eval()`
* Implementação manual das operações
* Maior controle da lógica
* Código mais seguro e organizado

---

## 🔹 V4.0 – Decimais, teclado e backspace

* Suporte a números decimais
* Suporte ao teclado
* Implementação de backspace (⌫)
* Bloqueio de múltiplas operações inválidas
* Mantida a lógica de uma operação por vez

---

## 🔹 V5.0 – Layout refinado e lançamento público

* Layout 4x4 reorganizado
* Botões padronizados
* Melhorias visuais com CSS
* Publicação via GitHub Pages
* Melhorias de acessibilidade

---

## 🔹 V5.1 – Rodapé do site

* Rodapé adicionado
* Links pessoais implementados
* Integração com GitHub e Instagram

---

# ⚠️ Limitações atuais

* Apenas uma operação por vez
* Não possui múltiplas operações encadeadas
* Não aplica precedência matemática
* Limitações intencionais para fins de aprendizado
